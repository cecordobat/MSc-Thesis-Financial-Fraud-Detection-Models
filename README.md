# MSc-Thesis-Financial-Fraud-Detection-Models

Master's Thesis: implementation of Deep Learning and Ensemble models for Financial Fraud Detection using Imbalanced Data Techniques

/////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////

+ Descripción del Proyecto
Este proyecto investiga la eficacia de distintos algoritmos de Machine Learning y Deep Learning para la detección de fraude financiero. El estudio se centra en el manejo de conjuntos de datos desbalanceados, la selección de características y la optimización de hiperparámetros para mejorar la precisión en la detección de transacciones fraudulentas.

+ Objetivos
Comparar modelos de clasificación tradicionales (Random Forest, XGBoost) frente a redes neuronales.
Implementar técnicas de sobremuestreo y submuestreo para tratar el desbalance de clases.
Evaluar el rendimiento de los modelos a partir de métricas de calidad robustas (AUC-ROC, F1-Score, Precision-Recall Curve, Confussion Matrix).

+ Tecnologías y Herramientas
Lenguaje: Python 3.9+
Librerías principales: Scikit-learn, TensorFlow, Keras, Pandas, NumPy, Matplotlib, Seaborn, Pytorch.
Entorno: Archivos python, Jupyter Notebook.

+ Estructura del Repositorio
/data: Scripts para carga y preprocesamiento de datos (los datasets pesados se excluyen del repo).
/notebooks: Análisis exploratorio de datos (EDA) y experimentos iniciales.
/src: Código fuente modular (limpieza, entrenamiento, evaluación).
/docs: Documentos relacionados con la tesis y artículos de investigación.
